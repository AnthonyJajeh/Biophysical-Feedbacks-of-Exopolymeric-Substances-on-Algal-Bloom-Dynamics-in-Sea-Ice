Authors: Anthony Jajeh, Kenneth Golden, Jody Deming, Jody Reimer.

All figures were generated using MATLAB and XPP, with the corresponding parameter values provided in the associated code.
The data obtained from XPP for the bifurcation plots were then passed into plotxppaut4p4.m to produce the current bifurcation figures.
The code for each figure is organized in its respective "Figure" folder.
Files containing analyses performed in Maple are identified by filenames beginning with "S".
